# Android Landing Page

A responsive landing page developed during the HTML and CSS course by Curso em Vídeo.

This project was built to practice responsive web design, semantic HTML, modern CSS techniques and layout organization.

- HTML5

- Clean semantic HTML structure

- Project organization
